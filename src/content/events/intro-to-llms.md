# Workshop Overview

Join us for a hands-on introduction to Large Language Models! This workshop is designed for students who are curious about how ChatGPT and other LLMs work under the hood.

## Topics Covered

- **Tokenization**: How text becomes numbers
- **Attention Mechanism**: The "secret sauce" of transformers
- **Prompt Engineering**: Getting the best outputs from LLMs
- **Practical Demo**: Building a simple chatbot using the OpenAI API

## Prerequisites

- Basic Python knowledge
- Laptop with Python 3.8+ installed
- Curiosity and enthusiasm!

## What to Bring

- Your laptop
- Questions you've always wanted to ask about AI

No prior machine learning experience required. We'll start from first principles!
